# Automated Form Submission System

## Project Overview
This project is an automation workflow built using n8n.

The system automatically:
- Collects form submissions
- Processes incoming data
- Stores responses into Google Sheets
- Reduces manual data entry

## Technologies Used
- n8n
- Google Sheets
- Webhooks
- APIs

## Workflow
Form Submission → n8n Workflow → Google Sheets

## Features
- Real-time form processing
- Automated data storage
- Easy workflow integration
- Beginner-friendly automation project

This project helped me learn:
- Workflow automation
- Webhooks
- API integrations
- Data handling using Google Sheets
